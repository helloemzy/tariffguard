## Project Context for Future Reference

### **Product Vision**

TariffGuard monitors tariff rate changes for import/export businesses, preventing costly surprises through automated alerts and impact calculations. The system targets SMBs ($1M-$50M import volume) underserved by enterprise solutions.

### **Tech Stack Decisions**

- **Frontend**: Next.js 14 + TypeScript + Tailwind CSS
- **Backend**: Supabase (PostgreSQL) with Row Level Security
- **Authentication**: Supabase Auth with social providers
- **Payments**: Stripe with subscription management
- **Hosting**: Vercel (frontend) + Supabase (backend)
- **Monitoring**: Sentry + PostHog analytics

### **Key Business Requirements**

- 99%+ alert accuracy through multi-source validation
- <6 hour alert delivery from government publication
- Support for unlimited HS codes in paid tiers
- Quote Helper for pre-quote rate validation
- Team collaboration features
- Mobile-responsive interface
